# 10 元文案快修

我可以帮你把一小段文案改得更清楚、更自然、更容易被读完。

## 价格

10 元 / 次

## 你发什么

- 原文：中文 300 字以内，或英文 120 词以内
- 用途：例如朋友圈、闲置转让、个人简介、简历、商品介绍
- 想要语气：自然、专业、短一点、有销售感、口语化等
- 必须保留的关键词

## 我交付什么

- 改写版 1 个
- 简短修改说明 3 条以内
- 一次轻微调整
